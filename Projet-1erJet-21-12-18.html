Les Données
¶

Le jeu de donnes contient les transactions effectues par cartes de crdit en septembre 2013 par les titulaires de carte europennes.
Cet ensemble de donnes prsente les transactions qui se sont produites en deux jours, o nous avons eu 492 fraudes sur 284 807 transactions.
L'ensemble de donnes est trs dsquilibr, les classes positives (fraudes) reprsentent 0,172% de toutes les transactions.
Il contient uniquement des variables d'entre numriques rsultant d'une transformation PCA. Les caractristiques V1, V2, ... V28 sont les composantes principales obtenues avec PCA, les seules caractristiques qui n'ont pas t transformes avec PCA sont 'Time' et 'Amount'.
La variable 'Time' contient les secondes coules entre chaque transaction et la premire transaction de l'ensemble de donnes. La variable 'Amount' est le Montant de la transaction, cette caractristique peut tre utilise pour l'apprentissage sensible aux cots dpendant de l'exemple. La fonction 'Class' est la variable de rponse et prend la valeur 1 en cas de fraude et 0 sinon. Compte tenu du rapport de dsquilibre de classes, nous recommandons de mesurer la prcision en utilisant l'aire sous la courbe de rappel de prcision (AUPRC). La prcision de la matrice de confusion n'est pas significative pour une classification non quilibre.

#AUCPR -> dOIT ETRE UTILIS SI classe dsquilibr
def AUCPRC(testy, probs,plot=F,withAUC=F,withACC=F):
    '''
    AUCPRC(
        true_labels [n_samples] ,
        probas [n_samples] ou decision function,
        plot=F
    )
    '''
    precision, recall, thresholds = precision_recall_curve(testy, probs)
    if plot:
        plt.plot([0, 1], [0.1, 0.1], linestyle='--')
        # plot the roc curve for the model
        plt.plot(recall, precision, marker='.')
        # show the plot
        plt.show()
    res = [("aucprc",auc(recall,precision))]
    if withAUC:
        res+=[("auc",roc_auc_score(testy, probs))]
    if withACC:
        cm=confusion_matrix(testy, probs)
        res+=[("acc",max(sum(np.diag(cm)),sum(np.diag(cm[::-1])))/np.sum(cm))]
    return res

def criticalCorr(n,alpha=0.05):
    df=n-2
    t=scipy.stats.t.ppf([alpha/2],df)[0];
    t2=t**2
    st2=sqrt(t2)
    return st2/sqrt(t2+df)

def pipelineLaunch():
    #LinearSVC,SVC -> long
    #penser a n_jobs=-1
    #ou autre solutions parralell
    algo=[LogisticRegression(),LinearDiscriminantAnalysis(),QuadraticDiscriminantAnalysis(),GaussianNB(),
         #KNeighborsClassifier()
          LinearSVC(),
          SVC(),
         DecisionTreeClassifier(),
         RandomForestClassifier()]
    z=pd.DataFrame()
    for i in tqdm(algo):
        i.fit(trainx,trainy)
        testP=i.predict(testx)
        trainP=i.predict(trainx)
        metricTrain=metrics(trainy,trainP)
        metrixTest=metrics(testy,testy)
        z.append(
            pd.DataFrame.from_records(metricTrain,columns=["metrics",repr(i.__class__).split(".")[-1]+"_train"],index="metrics")
        ).append(
            pd.DataFrame.from_records(metrixTest,columns=["metrics",repr(i.__class__).split(".")[-1]+"_test"],index="metrics")
        )
    return z
